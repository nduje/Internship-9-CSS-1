## Task

This project represents a **responsive recreation** of the following website:  
https://www.muji.eu/

The goal is to recreate the **layout and visual appearance** of the MUJI homepage using **HTML and CSS**, with focus on **responsiveness, hover effects, and navigation**.

## Requirements

- Navigation with announcement bar (marquee), icons, links and hover effects
- Responsive header layout (desktop & mobile)
- Highlights section with **horizontal scroll**
- Hero section with proper positioning and box model
- Best sellers section with hover effect and links
- One product section (**Most popular** or **Furniture / New arrivals**)
- Seasonal selection with full-width horizontal scroll
- MUJI stories section (4 items) with hover
- Fully recreated footer

## Setup

1. Clone or download the repository.
2. Open the project folder.
3. Open index.html in any modern web browser.
